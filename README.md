JPKIReader
====

マイナンバーカードの証明書エクスポート、署名、検証を行うライブラリ
Library that exports, signs, and verifies the certificate of JPKI card

## Description
- Windows 10 1803
- Visual Studio 2017
- C# Desktop Applicaion
- .Net Framework 4.6.1

## Requirement
- PaSoRi RC-S380
- JPKI Card
